Mattress Recommender

This repo was built to provide a recommendation for the best Mattress for an individual based on certain features. The model was built using 22,000+ reviews of various mattresses currently available. This repo contains the model, a notebook with a function built out to collect all of the review text data, the Flask web application, the exploratory data analysis. The culmination of this project is a deployed model available to use.

Test it out!



Synthesizer Price Predictor

This repo was built to provide an accurate price predictor for musical synthesizers. The model was built using 3,000 samples of sold synthesizer data from Reverb.com The repo contains the machine learning model, a notebook with a function built out to scrape and collect all of the data from Reverb.com, and an EDA Jupyter Notebook that provides an analysis of sales data.
